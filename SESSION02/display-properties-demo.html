Inline vs block elements

Block Elements

  1. Always occupy the full width of the screen
  2. Have a new line at the end of the element
  3. Example: p, heading tag, ul, li

Example of block elements

This is a paragraph which is a block element

Inline Elements

  1. Always occupy the width required to show the element's content
  2. Doesn't have a new line at the end of the element
  3. Example: a, img, input, button, label, span, strong
